Objetivo del proyecto Sistema de Gestion LiaStudio
Crear un sistema usable para el salon de belleza LiaStudio, implementando modulos profesionales:
Agenda de citas que se gestionan por categorias
Control de ingreso y gastos del negocio, para una mejor gestion financiera
Auth con JWT
El objetivo final es permitir al salón llevar un control eficiente de sus citas y finanzas desde un solo sistema.

Modulos principales
Autentificacion 
Agenda de citas
Categorias 
Clientes
Finanza 
Dockerixacion del sistema
